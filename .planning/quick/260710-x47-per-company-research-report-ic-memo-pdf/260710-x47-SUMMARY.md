# Quick 260710-x47: Per-Company Research Report (IC-Memo PDF) Summary

Branded on-demand IC-memo PDF rendered from stored `company_analysis` via a shared pdf-kit branding layer, with honest tags, code-computed comps, and zero LLM calls.

## What was built

- **Task 1 (`9617cf3`)** — `lib/reports/pdf-kit.ts`: brand palette (TEAL/INK/MUTED/GREEN/RED + AMBER/HAIRLINE), `drawLogo`, `wrapText` word-wrap, and `PdfCursor` (text/rule/bullet/sectionTitle/tag/paragraph/`ensureSpace` with auto page-break drawing the slim running header — small logo + company name + "RESEARCH MEMO · p. N" + hairline — on pages 2+). `digest.ts` refactored to import colors + `drawLogo` (import-only; behavior unchanged, all 299 pre-existing tests green).
- **Task 2 (`59e7116` RED, `8dcc02f` GREEN)** — `lib/reports/company-report-model.ts`: pure `buildCompanyReportModel(company, analysis, competitors, { generatedAt, stale })` → ordered section list (exec summary → business/moat → competitive landscape → valuation comps → analogue → outlook → IC conclusion), moat bars via `clampRating` (out-of-domain dropped, never clamped), comps rows exclusively from `buildCompsTable` with "—" null cells, IC badge mapping (strong_buy/buy→green, hold→muted, sell→red), `formatTag` (`[est · med]`, graceful degradation), `slugifyCompanyName`. 25 co-located Vitest tests.
- **Task 3 (`85609d6`)** — `lib/reports/company-report.ts` (server-only, no LLM calls): `buildCompanyReportPdf` draws the A4 memo (digest-geometry page-1 header with logo box 30, snapshot strip, IC badge, moat score bars, capability-matrix + comps tables, literal "illustrative comps, not a forecast" disclaimer, amber stale notice) and `generateCompanyReport` (ownership check `company.user_id === userId`, "run a deep dive first" on missing analysis, `isStale`, upload to `reports` bucket at `{userId}/{date}-{slug}-memo.pdf` upsert). Queries: `ReportFile.kind/label` (suffix-classified, digest date derivation untouched) + `listCompaniesWithAnalysis`. Action `generateCompanyReportNow` (auth guard → admin client → signed URL 3600s → revalidate). Reports UI: `CompanyReportCard` picker (disabled "no deep dive yet" options, stale badge, pending state, opens signed URL) with report list grouped Digests / Company memos; `GenerateMemoButton` on the company page header.
