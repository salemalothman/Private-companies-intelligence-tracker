Each table is authored as two exported/local types plus a registration entry:
```
export type XyzRow = { id: string; ...columns...; created_at: string; updated_at: string };
type XyzInsert = { id?: string; ...required cols...; optional?: ...; updated_at?: string };
// then inside `interface Database { public: { Tables: { ... } } }`:
xyz: { Row: XyzRow; Insert: XyzInsert; Update: Partial<XyzInsert> };
```
